Quick Start Guide

        	This is a guide to use MusicBot once it has already been assembled. Check the Schematics tab for images of MusicBot. In order to develop from scratch, take a look at the Development tab. 
        	This will be a plug-n-play style guide, although software installation will be required.

Materials

			MusicBot
				ESP8266-12E microcontroller
				NodeMCU motorshield
				FEETECH FS90R servo
				“Drumstick”
				“Drums”
			Windows 10 Computer
			Micro-USB cable
			Internet Connection

Part A: Arduino Setup

  1. Connect the ESP8266 to your computer via micro-USB cable.
  2. Open the Device Manager on your laptop. Under “Other Devices” there should be a device called CP120x USB to UART Bridge Controller. This is your hardware.
  3. Download the driver software from here: http://www.chinstruments.com/USB_416.shtml
  4. Right click on CP120x USB to UART Bridge Controller in your device manager. Click on Update Driver Software. Follow onscreen instructions to select and installed the drive you downloaded in the previous step.
  5. Download and install the Arduino IDE from here: https://www.arduino.cc/en/Main/Software
  6. Open up the Arduino IDE. Go to File > Preferences. Under Additional Boards Manager URL, copy and paste this address: http://arduino.esp8266.com/stable/package_esp8266com_index.json
  7. Go to Tools > Boards > Boards Manager. On the list, find “esp8266 of ESP8266 Community”. Select and install it.
  8. Go back to your device manager. Look under “Ports(COM & LPT)”. Find CP120x USB to UART Bridge Controller. Make note of the port number in parentheses.
  9. In Arduino, go to Tools > Board and select NodeMCU 1.0 (ESP8266-12E Module).
  10. Go to Tools > Port and select the port that was listed in your device manager.
  11. Go to Tools > Upload Speed and select 115200.
  12. You are now ready to run the programs included in this repository.

Part B: Operate in Autonomous Mode

  1. Connect the ESP8266 to the NodeMCU Motor Shield. Connect the servo to the GPIO of the motor shield. See the Schematics directory for more information and pictures.
  2. Download the Arduino program titled “MusicBot_Autonomous”.
  3. Setup the robot’s “drums” as detailed in the Schematic.
  4. Compile and upload the program by clicking Sketch > Upload or by clicking the right arrow at the top of the Arduino IDE.
  5. The robot will begin to play!

Part C: Operate with Human Control

  1. Connect the ESP8266 to the NodeMCU Motor Shield. Connect the servo to the GPIO of the motor shield. See the Schematics directory for more information and pictures.
  2. Download the Arduino program titled “MusicBot_Web”.
  3. Setup the robot’s “drums” as detailed in the Schematic.
  4. Compile and upload the program by clicking Sketch > Upload or by clicking the right arrow at the top of the Arduino IDE.
  5. Open your computer’s Network settings. Connect to the network named “MusicBot”. The password is “batman”. (The username and password is set in the Wifi.softAP function and can be set to your preference.)
  6. Open a browser window and navigate to http://192.168.4.1/A (If the above IP address does not work, open Tools > Serial Monitor. There will be an IP listed there.)
  7. There are three pre programmed “songs”. Clicking on the corresponding buttons will make the robot play each song!
